🎯 국내 최초 커스터마이징 시계 브랜드를 만들었습니다

참고할 선례 없는 환경에서 홍콩·일본 출장을 통해 글로벌 트렌드를 조사하고, 고객이 케이스·핸즈·밴드를 직접 선택하는 커스터마이징 시스템을 국내 최초로 기획·구축했습니다.

📚 매일 아침 2시간, '공부하는 MD'입니다

사수 없는 스타트업 환경에서 직무 공부를 꾸준히 이어왔습니다. Tableau로 상품 성과를 시각화하고, SQL·Python으로 데이터 분석 역량을 키워 데이터 기반 의사결정을 실천합니다.
